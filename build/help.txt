This page gives a simple description of the commands offered by this software:

l [n] -> go to absolute line n
fopen [buff] -> open a file with name 'buff' within the current directory
sq -> saves the current file and quit
s -> save the current file
dir [buff] -> sets the current directory of a window
w [n] -> opens n new windows
ninp -> open input file
nout -> open output file
help -> opens this page :)
